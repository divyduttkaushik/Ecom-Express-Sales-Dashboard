# Ecom Express Sales Dashboard | Power BI

An interactive Power BI dashboard developed to analyze e-commerce sales performance. This project demonstrates end-to-end business intelligence workflow, including data cleaning, data modeling, DAX calculations, KPI development, and interactive dashboard design to support data-driven decision-making.

---

## Project Objective

The objective of this project is to transform raw e-commerce sales data into meaningful business insights through an interactive dashboard that enables stakeholders to monitor sales performance, profitability, customer behavior, and regional trends.

---

## Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel

---

## Project Workflow

### 1. Data Preparation (Power Query)

- Imported raw sales data into Power BI
- Cleaned and transformed the dataset
- Handled missing values and inconsistent data
- Standardized data types and column formats

### 2. Data Modeling

- Built relationships between tables
- Designed an optimized data model for reporting

### 3. DAX Calculations

Created calculated measures and KPIs to analyze business performance.

Examples include:

- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Profit Margin

### 4. Dashboard Development

Designed an interactive dashboard with:

- KPI Cards
- Sales Trend Analysis
- Category-wise Sales
- Regional Performance
- Customer Insights
- Interactive Slicers and Filters

---

## 📈 Key Business Insights

- Identified high-performing product categories.
- Compared sales and profit across different regions.
- Analyzed sales trends over time.
- Highlighted customer purchasing patterns.
- Enabled quick monitoring of key business KPIs.
